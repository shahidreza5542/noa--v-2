## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/shahidreza5542/noa--v-2.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd noa--v-2
   ```
3. **Open `index.html` in your browser to get started.**
